Batalha Naval em C
Este é um projeto digital do clássico jogo Batalha Naval feito utilizando a linguagem C. O jogo foi desenvolvido para dois jogadores, com foco em praticar lógica de programação, manipulação de matrizes e controle de fluxo.

Objetivo
- Praticar conceitos de programação em C.
- Desenvolver um jogo interativo no terminal.
- Trabalhar com matrizes, estruturas de decisão e repetição.

Tecnologias
- Linguagem: C
- Bibliotecas utilizadas:
stdio.h, stdlib.h, locale.h
Como Jogar


1. Execute o programa.
2. No menu inicial, escolha "Inicia".
3. Cada jogador posiciona seus navios:
Porta-aviões (3 células)
Submarino (2 células)
Bote (1 célula)
4. Alternadamente, cada jogador faz ataques.
5. O jogo termina quando um dos jogadores destruir todos os navios do adversário.

 Estrutura
Dois tabuleiros: um para cada jogador.
Marcação: "X" → acerto   |    "O" → erro
Controle automático de turnos e verificação de vitória.

Funcionalidades
- Sistema de posicionamento de navios com validação.
- Rodadas de ataque alternadas.
- Exibição dos tabuleiros.
- Verificação de vitória.
